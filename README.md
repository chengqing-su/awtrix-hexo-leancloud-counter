# awtrix hexo leancloud

This an AWTRIX app to show hexo post read count.

Currently, it's only support show count.
In the feature, it will show comment count.

If you want to know how to develop an AWTRIX app, please follow [awtrixdocs](https://awtrixdocs.blueforcer.de/#/en-en/appcoding)

I also have a chinese version post to show this app.
